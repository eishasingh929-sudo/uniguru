# Run Entry Points

- Backend (Linux/macOS): `bash run/run_backend.sh`
- Backend (Windows): `powershell -ExecutionPolicy Bypass -File run/run_backend.ps1`
- Node middleware (Linux/macOS): `bash run/run_node.sh`
- Node middleware (Windows): `powershell -ExecutionPolicy Bypass -File run/run_node.ps1`
